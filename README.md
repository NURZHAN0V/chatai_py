# 🤖 Простой чат с нейросетью (OpenRouter)

Минимальный консольный бот на Python для общения с нейросетью через OpenRouter. Отвечает на ваши сообщения в реальном времени.

---

## 📋 Что нужно перед запуском

| Требование | Зачем |
|------------|-------|
| **Python 3** | Язык, на котором написана программа |
| **Ключ API от OpenRouter** | Ваш личный «пропуск» к нейросети ([получить здесь](https://openrouter.ai/)) |
| **Интернет** | Чтобы отправлять вопросы и получать ответы |

---

## ⚙️ Установка и запуск

1. **Установите библиотеку для работы с интернетом:**
   ```bash
   pip install requests
   ```

2. **Вставьте свой ключ в код:**
   ```python
   KEY = "sk-or-v1-..."  # Ваш ключ от OpenRouter
   ```

3. **Запустите программу:**
   ```bash
   python chat.py
   ```

---

## 💬 Как пользоваться

```
Ты: Привет, как дела?
Нейросеть: Привет! У меня всё хорошо, спасибо. Чем могу помочь?

Ты: Расскажи шутку
Нейросеть: Почему программисты путают Хэллоуин и Рождество? Потому что Oct 31 = Dec 25! 😄

Ты: выход  # ← напишите это, чтобы завершить работу
```

> 💡 Программа работает в цикле: пишите сколько угодно сообщений, пока не решите остановиться.

## ⚠️ Важные моменты

🔐 **Безопасность ключа**  
Никогда не показывайте и не передавайте свой `KEY` посторонним. Если ключ утечёт — им смогут воспользоваться другие, и с вашего счёта спишутся деньги.

🌐 **Нужен интернет**  
Без подключения к сети программа не сможет отправить запрос и покажет сообщение об ошибке.

🧠 **Модель можно менять**  
В строке `"model": "z-ai/glm-4.5-air:free"` можно указать любую другую модель из [каталога OpenRouter](https://openrouter.ai/models). Просто впишите её точное название.

---

## 🛠 Если что-то не работает

| Проблема | Возможная причина | Решение |
|----------|------------------|---------|
| `Ошибка 401` | Неверный или пустой ключ | Проверьте, что `KEY` вставлен правильно |
| `Ошибка 404` | Модель не найдена | Убедитесь, что название модели написано точно |
| `Ошибка 429` | Закончились бесплатные запросы | Подождите или добавьте средства на счёт |
| `ModuleNotFoundError: requests` | Не установлена библиотека | Выполните `pip install requests` |
| Программа «зависает» | Нет интернета или долгий ответ сервера | Проверьте подключение, подождите немного |
